## 📌 About the Project

Nomader is a fully responsive Tours & Travels website designed to provide users with an engaging and modern travel browsing experience. It showcases adventure activities, travel packages, and includes an interactive contact form.

---

## 🚀 Features

* 🌐 Fully responsive design (Mobile, Tablet, Desktop)
* 📱 Interactive mobile navigation menu
* 🎯 Smooth scrolling and dynamic header effects
* 🎨 Modern UI using Flexbox and CSS Grid
* 🧭 Adventure categories section with animations
* 🏝️ Travel packages with pricing details
* 📩 Contact form UI with clean layout

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **Font Awesome Icons**
